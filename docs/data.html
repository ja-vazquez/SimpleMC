Data¶

The data avaiable in SimpleMC is the following:

  • Supernovaes Type 1a: Union2.1, JLA and Pantheon.

  • Cosmic Chronometers: From https://arxiv.org/abs/1601.01701.

  • Baryonic Acoustics Oscillations: DR11LOWZ, DR11CMASS, 6dF Galaxy Survey, DR11 Lyman alpha and SDSS Main Galaxy Sample BAO.

  • Planck: WMAP9 and other reduced data.

Note

For a CPL model with curvature, we can test different combination of datasets avaliables in SimpleMC.
